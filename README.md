# SocialBunny

Making Social App By Hani

## Tech Stack

**Backend:** fastapi
**Frontend:** react

## Getting Started

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload  # For FastAPI
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
