# ⚛️ React Boilerplate

A modern React + Vite starter including:
- Emotion (CSS-in-JS)
- Framer Motion (animations)
- Lenis (smooth scrolling)
- Tailwind CSS (v3.3.x)

## Getting Started
```bash
npm install
npm run dev
```

## Tech
- React 18
- Vite 5
- Tailwind 3.3
- @emotion/react, @emotion/styled
- framer-motion
- lenis

## Structure
```
src/
  components/
    Hero.jsx
    Navbar.jsx
  hooks/
    useLenis.js
  styles/
    theme.js
  App.jsx
  main.jsx
  index.css
```
